나의 파이썬 모듈만들기

시작하며…

여러가지 토이프로젝트를 혼자서 진해할때 대부분을 python 으로 진행했습니다. (역시 파이썬이 혼자 공부하고 놀기에 좋습니다.)

그때마다 기존에 짰던 코드들을 짜집기해서 붙여넣고, 다시 이전코드를 분석 하느라, 간단한 취미용 프로젝트임에도 개발 속도가 나지 않더군요.

그래서, 그냥 쉽게 쉽게 쓸수있는 나만의 모듈을 만들어보고 싶었습니다.

어떤것들을 만들까?

대부분의 개인적인 취미나 생활에 필요한 것들을 만들다보니 다음의 기능들을 중복적으로 많이 사용하였습니다.

  1. db 접근
  2. 간단한 텔레그램 메신져 보내기
  3. 쓰레드 여러개 만들어서 처리하기
  4. 웹 파서 공용화하기
  5. 시간관련 계산하기
  6. 로그 남기기
  7. 각종 문자열 정규식
  8. 파일경로확인 및 싱크

위의 기능들을 그냥 간단히 최대한 간단히 라이브러리로… 이용해보고 싶습니다.

개인프로젝트에 대부분 사용할것이므로, 성능, 최적화의 이슈에서는 자유로울수 있을것 같습니다.

파이썬 공부겸해서, 취미용에 적합한 라이브러리를 만들어 봅니다.
